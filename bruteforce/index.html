URL verrouillée à décrypter par force brute

"Forcing brute" Un mot de passe est l'acte de le deviner par force brute - essayant littéralement tous les mots de passe jusqu'à ce que l'on fonctionne. Cette page est une simple application de force brute de preuve de concept. Link Lock est conçu pour décrypter une URL en essayant chaque mot de passe possible. Il n'est pas optimisé de quelque manière que ce soit et fait un minimum de vérification des erreurs.

Pour plus d'informations sur les URL de verrouillage des liens de forçage brute, lisez le problème ouvert Voir les problèmes sur GitHub à ce sujet.

Pour une application de force brute en ligne de commande qui effectue un traitement parallèle à l'aide de tous les noyaux du CPU, consultez ce projet.
